# NCAA Tournament Prediction Model

## 📊 Project Overview
This project aims to analyze NCAA men's basketball regular season data and identify which performance metrics most strongly influence tournament success (e.g., Sweet 16 and beyond).

## 📁 Data Sources
- Basic School Stats
- Advanced School Stats
- Opponent Stats
- NCAA Tournament Results

## ⚙️ Methodology
- Data cleaning and merging
- Feature selection based on basketball performance metrics
- Logistic Regression model to predict tournament outcomes

## 📈 Key Metrics Used
- 3P%
- 2P%
- Opponent FG%
- Offensive Rebound %
- Turnover %
- Defensive stats

## 🎯 Goal
Identify the most important factors teams should improve during the regular season to perform better in March Madness.

## 🧠 Tools
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook
